# Production Technical Track

This track defines technical production readiness for controlled deployments.

## Gate Command

```bash
make production-tech-check
```

Outputs:

- `evidence/production-tech/production-tech-check-<timestamp>.md`
- logs in `evidence/production-tech/logs-<timestamp>/`

## Included Controls

1. technical beta gate (`make beta-tech-check`)
2. stability trend gate (`scripts/tech-stability-report.sh`) with consecutive ready reports
3. registry external signer integration tests
4. production ops/SLO and incident runbook presence
5. upgrade/backward-compat/hardening playbook presence
